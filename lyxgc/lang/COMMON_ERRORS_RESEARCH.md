# Common Grammatical Errors, Typos & Misspellings (Regex-Detectable)

**Research compilation for lyx-gc `simple_rule` style.**  
Focus: Academic/technical writing. Format: `wrong → correct`.

## 1. Hungarian (hu)

| Wrong | Correct |
|-------|---------|
| wogóle | w ogóle |
| napewno | na pewno |
| naprawde | naprawdę |
| aggresszív | agresszív |
| abszolut | abszolút |
| két macskák | két macska |
| eljöttök (singular) | eljössz |
| „example" | „example" (correct: low-high „ ") |

**Notes:** Hungarian uses special quotation marks („ and "). Vowel length (a/á, e/é) and double consonants change meaning. Common phrase errors: "wogóle" (wrong) vs "w ogóle" (correct) – separate words.

## 2. German (de)

| Wrong | Correct |
|-------|---------|
| aufgrund dem | aufgrund des |
| dass (when "das" needed) | das |
| das (when "dass" needed) | dass |
| erfolgricher wie | erfolgreicher als |
| aus einander | auseinander |
| daß | dass |
| gemäß dem | gemäß des |
| trotz dem | trotz des |

**Notes:** Genitive vs dative with prepositions (aufgrund, wegen, trotz, während). "als" (inequality) vs "wie" (equality). "ss" vs "ß" confusion (post-reform).

## 11. Japanese (ja)

**Note:** Japanese uses Hiragana, Katakana, Kanji. Romanized/transliterated forms (rōmaji) commonly used in academic contexts for names, terms.

| Wrong (romanized) | Correct |
|-------------------|---------|
| watasi | watashi |
| zyun | jun |
| koukou | kōkō |
| tomodati | tomodachi |
| bennkyou | benkyō |
| nihongo | nihongo (correct) |
| kirei da | kirei (na-adjective) |
| kirei desu | kirei desu |

**Patterns (romanized):**
- Double n before b/m/p: ん → n (e.g. san-byaku → 300)
- Long vowels: ou→ō, uu→ū, ei→ē
- si→shi, ti→chi, tu→tsu, hu→fu

**Notes:** Most regex-detectable errors in academic writing appear in romanized citations, transcriptions. Kanji confusion (同/銅, 新/親) not easily regex-able. Particle は/が, に/で require context.

## 12. Chinese (zh)

**Note:** Simplified (CN) vs Traditional (TW/HK). Pinyin romanization used in academic citations.

| Wrong (pinyin) | Correct |
|----------------|---------|
| Zhōngguó | Zhōngguó (correct) |
| jianyi | jiànyì / jiǎnyì (tone marks) |
| shi ... de | shì ... de |
| zai ... zhong | zài ... zhōng |
| hen duo | hěn duō |
| bu shi | bù shì |
| yi ge | yī gè |

**Common typos (pinyin):**
- Tone mark omission: "hen" → "hěn"
- ng/n confusion: "zong" vs "zhong"
- zh/ch/sh vs z/c/s
- iu vs iou, ui vs uei

**Notes:** Regex works best on romanized text. Character-level errors (错别字) like 的/地/得, 在/再 require character analysis.

## 13. Arabic (ar)

**Note:** Right-to-left script. Romanized (transliterated) forms used in Western academic writing.

| Wrong (transliterated) | Correct |
|------------------------|---------|
| al- (sun letters) | assimilated: ash-shams, ar-rajul |
| al-kitab | al-kitāb (long ā) |
| Mohammed | Muḥammad |
| Quran | Qurʾān |
| ibn | ibn (not "bin" in formal) |

**Common errors (in Arabic script):**
- ة vs ه (ta marbuta vs ha)
- ا vs آ (alif vs alif madda)
- همزه placement
- Definite article ال + sun letters

**Notes:** Many errors are grammatical (agreement, word order) not easily caught by simple regex. Transliteration standards vary (ALA-LC, ISO, etc.).

## 14. Hindi (hi)

**Note:** Devanagari script. Romanized (IAST/Hinglish) common in technical/academic contexts.

| Wrong (romanized) | Correct |
|-------------------|---------|
| pariksha | parīkṣā |
| ladaka achi hai | ladakā achā hai |
| main gayi hoon | main gayā hoon |
| mein school | school mein |
| ho tel mein | hotel mein |
| puchhe gaye | puchhe gaye (gender agreement) |
| vivaran puche gaye | vivaran puche gaye |

**Patterns:**
- Matra/diacritic: short i (इ) vs long ī (ई), short u vs ū
- ai vs e, au vs o
- Retroflex vs dental: t vs ṭ, d vs ḍ
- Postposition placement: noun + postposition (school mein)

**Notes:** Gender agreement (masculine/feminine) affects verb forms. Script-level errors need character analysis.

## Summary: Regex-Friendly vs Context-Dependent

**Best for regex (wrong→correct patterns):**
- Hungarian, German, Polish, Czech: many fixed phrase/word errors
- Dutch, Swedish: compound splitting, de/dem
- Russian: specific wrong words (вообщем, придти)
- Italian, Spanish, Portuguese: homophones, apostrophes
- Greek: specific misspellings

**Harder for simple regex:**
- Japanese, Chinese: script + romanization variants
- Arabic: script + transliteration standards
- Hindi: script + matra/retroflex

## Implementation Suggestion

For each language module, add `simple_rule(wrong, correct)` entries. Example (hu.py):

```python
simple_rule("wogóle", "w ogóle"),
simple_rule("napewno", "na pewno"),
```

For languages with non-ASCII, ensure tokenizer handles the script. Romanized forms (ja, zh, ar, hi) can share patterns across Latin-script preprocessing stages.
